# Crypto Sentinel 🤖📈

Bot de surveillance crypto analysant tweets et news pour détecter des opportunités.

## Fonctionnalités
- 🕵️ Analyse de sentiment en temps réel
- 📊 Surveillance multi-crypto (RUNE, ATOM, DOT)
- 🔔 Notifications Telegram automatisées
